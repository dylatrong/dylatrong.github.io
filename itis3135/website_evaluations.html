Website Evaluations

Arngren.net

Arngren is a Norwegian gadget retailer website that is infamous in the web design community. It functions as a digital catalog but lacks almost all modern structural elements, looking more like a collage than a store.

  • Contrast: The site fails significantly here; blue text links are placed over busy, multi-colored images, making them nearly impossible to read.
  • Repetition: There is no consistent font hierarchy or color palette. Every element competes for attention with different sizes and colors.
  • Alignment: The layout ignores grid systems entirely. Images and text blocks are scattered randomly rather than aligned to a common edge.
  • Proximity: There is zero whitespace or grouping. Toys, car parts, and electronics are mashed together, making it difficult to tell which price tag belongs to which item.

Assessment: This site is a textbook example of poor design because it ignores alignment and proximity, resulting in a cognitive overload for the user.

Stripe

Stripe is a financial infrastructure platform known for setting high standards in modern web UI/UX, focusing on developer experience and clarity.

  • Contrast: The site uses sharp contrast between the dark text and clean white background (or dark mode equivalents), as well as vibrant accent colors for call-to-action buttons.
  • Repetition: They utilize a consistent "glassmorphism" blur effect, specific font weights, and a unified color palette across all sections.
  • Alignment: Every element, from the navigation bar to the feature lists, snaps to a strict grid, guiding the user's eye naturally down the page.
  • Proximity: Generous padding and margins separate different product features, while related headlines and subtext are grouped tightly together.

Assessment: Stripe effectively uses proximity and alignment to create a layout that feels organized, professional, and easy to navigate.
